# Housing-data-analysis
Housing Data Analysis is a project using Python, NumPy, and Pandas. This project involves data cleaning, handling missing values, filtering data, grouping, sorting datasets, and performing correlation analysis on a real-world housing dataset.
It was completed as part of Data Analytics Bootcamp by TECHYSPOT.
📌 Tasks Performed
Count missing values in each column.
Drop rows containing missing values.
Replace missing values using appropriate methods.
Filter rows where:
median_income > 5
housing_median_age > 30
Compute the average house value by ocean_proximity.
Sort the dataset by median_income and display the top 10 rows.
Find the correlation between median_income and median_house_value.
